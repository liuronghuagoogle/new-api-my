Claude Code 中转站

安装指南

快速配置 Claude Code CLI 工具，开始使用 AI 编程助手

1

检查 Git 安装

Git 是版本控制工具，Claude Code 依赖它来管理代码。首先验证系统是否已安装。

PowerShell / CMD 
git --version
如果显示版本号：说明已安装，可跳过此步骤
如果提示未找到命令：请按以下方式安装

安装 Git

  • 方法1：访问 Git 官网 下载安装包，按默认选项安装
  • 方法2：使用提供的离线安装包
2

安装 Node.js

Node.js 是 JavaScript 运行环境，Claude Code 基于此构建。要求版本 ≥ 18

重要提示：Node.js 版本必须大于或等于 18。如果已安装旧版本，请升级到最新 LTS 版本。

安装方式

  • 方法1：访问 Node.js 官网，下载 LTS 版本的 .msi 安装包
  • 方法2：使用提供的离线安装包
3

验证 Node.js 安装

确认 Node.js 和 npm 已正确安装

PowerShell / CMD 
node --version
npm --version
PowerShell 执行策略问题：
如果 PowerShell 在执行 npm.ps1 时被拦截，请先运行以下命令：
PowerShell (管理员权限) 
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
4

安装 Claude Code

使用 npm 全局安装 Claude Code CLI 工具

PowerShell / CMD 
npm install -g @anthropic-ai/claude-code --registry=https://registry.npmmirror.com
命令使用国内镜像源 (npmmirror.com) 以提高下载速度
5

配置环境变量

设置中转站地址和 API 密钥，三种方式任选其一

临时配置（仅当前会话）- PowerShell

PowerShell 
$env:ANTHROPIC_BASE_URL = "https://ai.facai.cloudns.org"
$env:ANTHROPIC_AUTH_TOKEN = "你的API密钥"
请将 你的API密钥 替换为在管理台 API 密钥管理中创建的实际密钥

永久配置（用户级）- PowerShell

PowerShell 
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL","https://ai.facai.cloudns.org",[System.EnvironmentVariableTarget]::User)
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN","你的API密钥",[System.EnvironmentVariableTarget]::User)

永久配置（用户级）- CMD

CMD 
setx ANTHROPIC_BASE_URL "https://ai.facai.cloudns.org"
setx ANTHROPIC_AUTH_TOKEN "你的API密钥"
重要：配置完成后，需要重新打开 PowerShell 或 CMD 窗口，环境变量才会生效
6

VSCode 插件配置

如果使用 VSCode 的 Claude 插件，需要配置 config.json 文件

配置文件路径

C:\Users\你的用户名\.claude\config.json （如果文件不存在，请手动创建）

JSON 
{
  "primaryApiKey": "你的API密钥",
  "customBaseUrl": "https://ai.facai.cloudns.org"
}
7

验证安装

确认 Claude Code 已正确安装并配置

PowerShell 验证

PowerShell 
claude --version
echo $env:ANTHROPIC_BASE_URL
echo $env:ANTHROPIC_AUTH_TOKEN

CMD 验证

CMD 
claude --version
echo %ANTHROPIC_BASE_URL%
echo %ANTHROPIC_AUTH_TOKEN%
验证成功标志：
• claude --version 显示版本号
• BASE_URL 显示 https://ai.facai.cloudns.org
• AUTH_TOKEN 显示您的 API 密钥
1

检查 Git 安装

macOS 通常已预装 Git，首先验证是否存在

Terminal 
git --version
如果显示版本号：说明已安装，可跳过此步骤
如果提示未找到命令：请按以下方式安装

安装 Git

Terminal (使用 Homebrew) 
brew install git

如果没有 Homebrew，请先访问 brew.sh 安装

2

安装 Node.js

使用 Homebrew 或官网安装 Node.js。要求版本 ≥ 18

方法1：使用 Homebrew（推荐）

Terminal 
brew install node

方法2：官网下载

访问 Node.js 官网，下载 macOS 安装包 (.pkg)

3

验证 Node.js 安装

确认 Node.js 和 npm 已正确安装

Terminal 
node --version
npm --version
4

安装 Claude Code

使用 npm 全局安装 Claude Code CLI 工具

Terminal 
npm install -g @anthropic-ai/claude-code
5

配置环境变量

设置中转站地址和 API 密钥

临时配置（仅当前会话）

Terminal 
export ANTHROPIC_BASE_URL="https://ai.facai.cloudns.org"
export ANTHROPIC_AUTH_TOKEN="你的API密钥"

永久配置（Shell 配置文件）

根据使用的 shell（bash 或 zsh），将以下内容添加到对应配置文件：

~/.bash_profile 或 ~/.zshrc 
export ANTHROPIC_BASE_URL="https://ai.facai.cloudns.org"
export ANTHROPIC_AUTH_TOKEN="你的API密钥"

保存后执行以下命令使配置生效：

Terminal 
source ~/.zshrc  # 或 source ~/.bash_profile
请将 你的API密钥 替换为在管理台 API 密钥管理中创建的实际密钥
